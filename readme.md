# Ordem recomendada:

### Se você estiver configurando os serviços do zero, a ordem recomendada para você fazer as configurações com base na dificuldade de implementar o serviço e nas suas dependências é:

- ip
- dchp
- dns
- apache
- samba
- ftp
- ssh
