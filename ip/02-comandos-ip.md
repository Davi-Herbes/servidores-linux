# Comandos úteis

## Lista ips e outras coisas

```shell
ip addr

ip a s
```

## Rotas gateways e placa de rede

```shell
ip route
```

## Ver o estado do ip

```shell
ip link

ip -s link # Mostra também o fluxo de dados
```

## Ver portas abertas

```shell
ss # Mostra todas

ss -lt # Mostra as abertas
```
