# Configurando Apache: Introdução

## Instalação

```bash
apt install apache2
```

## Localizações dos arquivos

### Configurações apache

```bash
cd /etc/apache2
```

### Aplicação do site

```bash
cd /var/www/html
```

### Sites

```bash
cd /etc/apache2/sites-available

# Ver os arquivos:
ls -lah
```
